# Inteligencia de Datos y RAG Avanzado

Formación de **ONE AI for Tech** orientada a la construcción de aplicaciones inteligentes conectadas a datos reales mediante inteligencia artificial, técnicas de **Retrieval-Augmented Generation (RAG)** y automatización del análisis de datos.

El recorrido combina modelos de lenguaje, recuperación de información, agentes, bases de datos vectoriales y herramientas para desarrollar interfaces y aplicaciones utilizables en escenarios prácticos.

## Objetivo de la formación

Aprender a diseñar y construir aplicaciones de inteligencia artificial capaces de buscar, interpretar y utilizar información proveniente de fuentes de datos reales.

La formación aborda la creación de sistemas RAG, el uso de agentes para análisis de datos y el desarrollo de aplicaciones que integran recuperación de información, modelos de lenguaje e interfaces accesibles.

## Objetivos de aprendizaje

Al completar la formación, se espera poder:

- comprender los fundamentos de Retrieval-Augmented Generation;
- conectar modelos de lenguaje con fuentes externas de información;
- trabajar con documentos PDF y archivos CSV;
- utilizar embeddings para representar y recuperar información;
- estructurar búsquedas mediante bases de datos vectoriales;
- crear agentes con memoria y capacidad de interacción;
- integrar herramientas como LangChain y LangGraph;
- automatizar tareas de análisis y transformación de datos;
- generar informes y visualizaciones interactivas;
- desarrollar interfaces simples con Streamlit;
- preparar aplicaciones para su publicación en la nube.

## Seguridad

Los proyectos que utilicen modelos, APIs o servicios externos deberán gestionar sus secretos mediante variables de entorno.

No deben versionarse:

```text
.env
.venv/
__pycache__/
.ipynb_checkpoints/
*.db
*.db-shm
*.db-wal
```

Antes de publicar notebooks, bases de conocimiento o ejemplos de datos, se deberá comprobar que no contengan:

- claves o tokens;
- información personal;
- rutas locales;
- documentos privados;
- datos sensibles;
- outputs innecesariamente extensos;
- mensajes de error que expongan credenciales.
