# Ingeniería de Agentes y Automatización con IA

Esta formación forma parte del programa **Tech AI Builder** de **ONE AI for Tech**.

El objetivo de la formación es aprender a crear agentes inteligentes y automatizar procesos utilizando herramientas de Inteligencia Artificial, modelos de lenguaje, LangChain, LangGraph y n8n.

## Sobre la formación

Esta formación está orientada a la construcción de soluciones más completas y automatizadas con IA.

A lo largo del recorrido se trabajan agentes capaces de razonar, tomar decisiones, utilizar herramientas externas y ejecutar tareas de forma autónoma. También se exploran flujos de automatización con n8n, integración de APIs, lógica condicional, manejo de errores, monitoreo y creación de pipelines inteligentes.

La formación combina conceptos de **agentes de IA**, **orquestación de LLMs**, **automatización de procesos**, **workflows inteligentes** e **integraciones entre servicios**.

## Objetivos de aprendizaje

Al finalizar esta formación, el objetivo es haber desarrollado una base práctica para:

* crear agentes inteligentes capaces de razonar y tomar decisiones;
* utilizar LangChain y LangGraph para modelar flujos de IA;
* implementar ciclos de razonamiento y acción, como ReAct;
* orquestar múltiples agentes trabajando de forma coordinada;
* integrar LLMs con herramientas externas;
* aplicar técnicas como web scraping y gestión de estado;
* construir automatizaciones con n8n;
* integrar servicios como correo electrónico, hojas de cálculo y APIs;
* aplicar IA dentro de flujos automatizados;
* manejar errores, notificaciones y monitoreo;
* construir soluciones más robustas y cercanas a escenarios reales.

## Organización de la formación

La formación se organiza en dos bloques principales:

1. **Construye agentes inteligentes con LangGraph y LLMs**
2. **Crea flujos automatizados e integraciones inteligentes con n8n**

## Paso 01 — Construye agentes inteligentes con LangGraph y LLMs

En esta etapa se trabaja la creación de agentes inteligentes capaces de razonar, tomar decisiones y utilizar herramientas para ejecutar tareas.

Se utilizan recursos como **LangChain** y **LangGraph** para modelar flujos de IA como grafos, implementar ciclos de pensamiento y acción, y coordinar múltiples agentes.

También se abordan integraciones con herramientas externas, técnicas de web scraping, control de estado, gestión de información y construcción de aplicaciones inteligentes con interfaces interactivas.

## Paso 02 — Crea flujos automatizados e integraciones inteligentes con n8n

En esta etapa se trabaja la creación de automatizaciones completas utilizando **n8n**.

El foco está en construir flujos visuales, integrar servicios externos, conectar APIs y aplicar lógica condicional para automatizar procesos del día a día.

También se explora el uso de Inteligencia Artificial dentro de las automatizaciones, incluyendo análisis de datos, clasificación de información, generación de respuestas automatizadas y construcción de flujos más avanzados con manejo de errores, notificaciones y monitoreo.
