# Gemini y Python: orquestando LLMs con LangChain

Curso perteneciente al programa **Tech AI Builder**, dentro de la formación **Desarrollo y Orquestación con IA Generativa**.

Este curso está orientado a la creación de soluciones con **Python**, **Gemini** y **LangChain**, explorando cómo orquestar modelos de lenguaje, estructurar prompts, procesar salidas, trabajar con datos estructurados y construir agentes capaces de utilizar herramientas.

## Objetivo del curso

El objetivo del curso es aprender a utilizar **Python** y **Gemini** junto con **LangChain** para construir aplicaciones basadas en modelos de lenguaje.

A lo largo del curso se trabaja la integración de LLMs, la creación de cadenas de ejecución, el uso de prompt templates, el procesamiento de salidas estructuradas, el análisis de imágenes y la creación de agentes orquestadores capaces de elegir herramientas según la tarea.

## ¿Por qué realizar este curso?

Este curso permite:

* aprender a utilizar Python y Gemini para explorar el potencial de LangChain;
* comprender cómo orquestar diferentes LLMs e integrar soluciones inteligentes;
* crear agentes capaces de analizar imágenes y proporcionar respuestas precisas;
* estructurar prompts reutilizables y salidas en formatos controlados;
* construir herramientas para que los agentes puedan resolver tareas específicas;
* mejorar la productividad mediante herramientas de IA aplicadas al desarrollo.
