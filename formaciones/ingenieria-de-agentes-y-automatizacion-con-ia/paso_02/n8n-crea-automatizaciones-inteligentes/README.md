# N8N: Crea automatizaciones inteligentes

Curso perteneciente al programa **Tech AI Builder**, dentro de la formación **Ingeniería de Agentes y Automatización con IA**.

Este curso está orientado a la creación de automatizaciones con **n8n**, combinando servicios externos, APIs, herramientas de Google y modelos de Inteligencia Artificial dentro de flujos de trabajo visuales.

## Objetivo del curso

El objetivo del curso es aprender los fundamentos de n8n y utilizarlos para construir flujos de automatización que conecten servicios, procesen información y ejecuten acciones de forma automática.

A lo largo del curso se trabaja con triggers, APIs, modelos de IA, condiciones, estructuras JSON, código dentro de n8n e integraciones con herramientas como Gmail, Google Drive y Google Cloud.

## ¿Por qué realizar este curso?

Este curso permite:

* comprender los fundamentos para crear workflows en n8n;
* configurar triggers personalizados para servicios como Gmail;
* conectar y utilizar APIs para integrar información de la web;
* incorporar modelos de Inteligencia Artificial en automatizaciones;
* construir flujos condicionales;
* trabajar con datos en formato JSON;
* ejecutar código dentro de n8n;
* conectar workflows con Google Drive;
* configurar servicios mediante Google Cloud Console;
* crear automatizaciones adaptadas a necesidades concretas.

## Prácticas y workflows

Este curso incluye la creación de workflows con n8n, conexiones con APIs, integraciones con servicios externos y automatizaciones asistidas por IA.

Los archivos exportados de n8n y otros recursos prácticos relevantes podrán organizarse en:

```txt
workflows/
```

Los workflows pueden guardarse como archivos `.json`, siempre que hayan sido revisados previamente para comprobar que no contengan:

* API keys;
* tokens de acceso;
* credenciales;
* identificadores sensibles;
* direcciones de correo personales;
* datos privados utilizados durante las pruebas.

Los archivos adicionales, scripts o ejemplos de código podrán ubicarse en:

```txt
codigo/
```

Estas carpetas solo se crearán si el contenido generado durante el curso resulta útil para el repositorio.

## Seguridad

Las credenciales y claves utilizadas por los workflows no deben incluirse en el repositorio.

Antes de publicar una exportación de n8n, se debe comprobar que las conexiones estén referenciadas mediante credenciales configuradas en la plataforma y que no existan secretos escritos directamente dentro de nodos, expresiones o parámetros.
