# Tech AI Builder

Repositorio personal para documentar mi avance en el programa **Tech AI Builder**, parte de **ONE AI for Tech**.

El programa está orientado al desarrollo de habilidades vinculadas con **Inteligencia Artificial**, **automatización**, **agentes**, **RAG**, **datos** y **Oracle Cloud Infrastructure**.

## Sobre el programa

**Tech AI Builder** reúne distintas formaciones enfocadas en Inteligencia Artificial y Cloud. Según el plan de estudios, el programa contempla **18 cursos** y una carga estimada de **154 horas**.

Las formaciones no tienen un orden obligatorio y no es necesario completarlas todas para avanzar. Sin embargo, mi objetivo personal es recorrer y completar todas las formaciones disponibles.

## Formaciones

El programa está compuesto por las siguientes formaciones:

* **Nivelación**
* **Desarrollo y Orquestación con IA Generativa**
* **Ingeniería de Agentes y Automatización con IA**
* **Inteligencia de Datos y RAG Avanzado**
* **Oracle Cloud Infrastructure**

## Objetivo del repositorio

El objetivo de este repositorio es organizar y documentar el proceso de aprendizaje a lo largo del programa.

En particular, busca:

* Registrar el avance por formación, paso y curso.
* Documentar los conceptos principales trabajados.
* Organizar apuntes, recursos y ejercicios relevantes.
* Mantener una referencia personal clara para futuras consultas.
* Mostrar el proceso de aprendizaje de forma prolija en GitHub.

## Criterio de organización

El repositorio se organiza principalmente por **formación**.

Dentro de cada formación se documentan los pasos, cursos, artículos, recursos y certificados correspondientes.

Cada curso puede incluir, según corresponda:

* un `README.md` con el resumen del curso;
* apuntes o notas personales;
* recursos utilizados;
* ejercicios o prácticas realizadas;
* certificado del curso completado.

Los certificados se guardan dentro de la carpeta del curso correspondiente, ya que el programa entrega certificados por **curso completado**, no necesariamente por formación completa ni por finalización total del programa.

## Notas sobre el contenido

Este repositorio está pensado como una documentación personal de estudio.

Cuando corresponda, se incluyen referencias a artículos, documentación pública o recursos oficiales. No se incluyen materiales privados, restringidos o innecesarios para comprender el avance del programa.
