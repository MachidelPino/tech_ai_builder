# IA para Desarrolladores: Desarrollando Códigos con ChatGPT, Grok, Claude y Gemini

Curso perteneciente al programa **Tech AI Builder**, dentro de la formación **Desarrollo y Orquestación con IA Generativa**.

Este curso está orientado al uso de herramientas de Inteligencia Artificial Generativa como apoyo en tareas de programación, generación de código, refactorización, validación, documentación y construcción de prototipos.

* **Programa:** Tech AI Builder

* **Categoría:** IA para Programación
* **Plataforma:** Alura
* **Estado:** Pendiente
* **Certificado:** Pendiente

## Objetivo del curso

El objetivo del curso es explorar cómo distintas herramientas de IA pueden asistir el trabajo de desarrollo de software, desde la generación inicial de código hasta la mejora, adaptación, documentación y validación de soluciones.

A lo largo del curso se trabajan herramientas como **ChatGPT**, **Gemini**, **Claude**, **Grok**, **NotebookLM**, **CodePen** y **Google Colab**, aplicadas a escenarios prácticos de programación.

## ¿Por qué realizar este curso?

Este curso permite:

* explorar el uso de ChatGPT, Claude, Grok y Gemini para optimizar el desarrollo de código;
* aplicar técnicas de refactorización y validación de código con asistencia de IA;
* mejorar la generación de prototipos funcionales;
* crear prompts más precisos para obtener respuestas útiles de herramientas de IA;
* experimentar con entornos como CodePen y Google Colab;
* utilizar Python y JavaScript para pruebas, ajustes y mejoras de código.

## Detalle de aulas

### Aula 01 — Generación de Código

Contenidos principales:

* Presentación del curso.
* Preparación del ambiente.
* Uso de métodos tradicionales para resolver problemas de código.
* Uso de Canvas en Gemini y ChatGPT.
* Buenas prácticas para crear prompts.
* Herramientas de desarrollo colaborativo.
* Optimización de respuestas con IA.
* Creación de resúmenes inteligentes con NotebookLM.
* Comparación entre asistentes y copilotos.

### Aula 02 — De la demanda al prototipo

Contenidos principales:

* Exploración del Canvas de ChatGPT.
* Exploración del Canvas de Gemini.
* Refactorización de código.
* Creación de una pantalla de comentarios.
* Mejora de recopilación de comentarios en un proyecto.
* Transformación de un prompt en un prototipo con IA.
* Uso de NotebookLM como apoyo para resumir y organizar información.

### Aula 03 — Ejecución de Código

Contenidos principales:

* Ejecución de código en distintos lenguajes.
* Filtros de contenido.
* Personalización de servicios.
* Uso de Google Colab.
* Prácticas de prompting.
* Relación entre IA y soluciones aplicadas a problemas reales.
* Uso de NotebookLM para resumir y consolidar aprendizajes.

### Aula 04 — Conociendo a Claude y Grok

Contenidos principales:

* Introducción a Claude y Grok.
* Migración de código entre lenguajes.
* Adaptación de código para diferentes entornos.
* Comparación de herramientas de IA para desarrolladores.
* Uso de NotebookLM para organizar información sobre herramientas de IA.

### Aula 05 — Corrección del código y buenas prácticas

Contenidos principales:

* Pruebas de código en distintos entornos.
* Creación de documentación.
* Documentación de código.
* Buenas prácticas en proyectos asistidos por IA.
* Uso de herramientas complementarias.
* Integración de IA como copiloto de desarrollo.
* Cierre del curso.
