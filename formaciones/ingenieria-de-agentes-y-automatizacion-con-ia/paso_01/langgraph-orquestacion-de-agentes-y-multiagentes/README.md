# LangGraph: Orquestación de agentes y multiagentes

Curso perteneciente al programa **Tech AI Builder**, dentro de la formación **Ingeniería de Agentes y Automatización con IA**.

Este curso está orientado a la creación y orquestación de agentes inteligentes utilizando **LangGraph**, con foco en flujos de decisión, agentes ReAct, herramientas externas, búsqueda web, web scraping, persistencia, streaming, human-in-the-loop y sistemas multiagente.

## Objetivo del curso

El objetivo del curso es aprender a construir agentes inteligentes capaces de combinar razonamiento y acción, utilizar herramientas externas, mantener estado, ejecutar búsquedas, realizar web scraping y coordinar múltiples agentes dentro de flujos más complejos.

A lo largo del curso se trabaja con **LangGraph** para modelar flujos mediante nodos, aristas y decisiones condicionales, construyendo soluciones más robustas y cercanas a casos reales.

## ¿Por qué realizar este curso?

Este curso permite:

* desarrollar agentes inteligentes que integren pensamiento y acción;
* automatizar interacciones en sistemas de chat y procesos de recolección de datos;
* implementar estrategias de ciclo de pensamiento y acción mediante flujos de control;
* crear grafos de decisión con nodos y aristas condicionales;
* configurar almacenamiento seguro de API keys y gestión de estados con SQLite;
* integrar técnicas de web scraping con Selenium y BeautifulSoup para análisis de datos.
