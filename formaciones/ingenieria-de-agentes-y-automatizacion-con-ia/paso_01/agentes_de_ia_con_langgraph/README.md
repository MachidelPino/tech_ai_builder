# Agentes de IA con LangGraph

Curso perteneciente al programa **Tech AI Builder**, dentro de la formación **Ingeniería de Agentes y Automatización con IA**.

Este curso está orientado a la creación de agentes de IA utilizando **LangChain**, **LangGraph** y modelos de lenguaje como **Google Gemini**. El objetivo es construir flujos donde los agentes puedan razonar, utilizar herramientas, realizar búsquedas y coordinarse mediante grafos.

## Objetivo del curso

El objetivo del curso es aprender a construir agentes inteligentes capaces de utilizar LLMs, herramientas externas y flujos de trabajo basados en grafos.

A lo largo del curso se trabaja la configuración de modelos de lenguaje, la creación de agentes con herramientas, la realización de búsquedas, la combinación de múltiples agentes y la construcción de una visualización interactiva para presentar el resultado final.

## ¿Por qué realizar este curso?

Este curso permite:

* usar LangChain para crear cadenas de procesamiento con LLMs como Google Gemini;
* equipar LLMs con herramientas para resolver tareas prácticas, como búsquedas en la web;
* implementar agentes autónomos basados en ReAct;
* modelar flujos de trabajo de IA como grafos con LangGraph;
* crear sistemas multiagente usando enrutadores y supervisores;
* publicar una aplicación con una interfaz web interactiva usando Gradio.

## Detalle de aulas

### Aula 01 — Configurando el LLM

Contenidos principales:

* Presentación del curso.
* Preparación del ambiente de trabajo.
* Sitios web y bibliotecas necesarias.
* Configuración de APIs.
* Conexión con un modelo de lenguaje.
* Primeras pruebas con un LLM.

### Aula 02 — Haciendo las primeras búsquedas

Contenidos principales:

* Uso de LLMs con herramientas externas.
* Creación del primer agente de IA.
* Uso de prompt templates en LangChain.
* Construcción de un agente capaz de realizar búsquedas.
* Primeras prácticas con agentes y herramientas.

### Aula 03 — Perfeccionando búsquedas con múltiples agentes

Contenidos principales:

* Búsqueda de información científica.
* Definición de un grafo para combinar agentes.
* Creación de un nodo supervisor para coordinar respuestas.
* Orquestación de agentes.
* Manejo de estado en workflows de grafos.
* Integración de agentes reactivos.

### Aula 04 — Construyendo una visualización

Contenidos principales:

* Creación de una visualización.
* Presentación del flujo construido.
* Cierre del curso.
